Not sure which ZGeode to pick? Try 64-bit first — it covers almost every phone made in the last several years. If it fails to install or crashes on open, fall back to the 32-bit build.

Install in order

  1. 1 Install Zenith GDPS This is the actual game. Allow installs from unknown sources if prompted.
  2. 2 Install ZGeode Pick the 64-bit build unless told otherwise. This is the launcher + mod loader.
  3. 3 Open ZGeode, not Zenith GDPS directly ZGeode launches the game with mods active and already pointed at our server.

What this actually is

ZGeode is a retargeted build of the open-source Geode Android launcher — same mod-loading engine, pointed at Zenith's servers instead of the official Geometry Dash backend.

Source is public — view it on GitHub, or browse all release files directly.

Zenith is an independent fan project, unaffiliated with RobTop Games or the Geode team. Geometry Dash is the property of RobTop Games.
